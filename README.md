## Actividades

### Actividad 1: Sistema de Gestión de Estaciones de Servicio

Sistema Spring Boot empresarial para gestionar estaciones de servicio consumiendo APIs del Ministerio de Energía español.

**Características:**
- ✅ Arquitectura Hexagonal (Puertos y Adaptadores)
- ✅ Validación XML contra esquemas XSD con JAXB
- ✅ Persistencia con PostgreSQL y JPA
- ✅ Spring Batch para procesos ETL
- ✅ API REST documentada con Swagger/OpenAPI
- ✅ Consultas geográficas optimizadas

Ver [actividad1/README.md](actividad1/README.md) para más detalles.

## Información General

Este repositorio contiene las actividades prácticas de la asignatura Base de Datos Avanzadas, donde se trabajan conceptos de:

- Arquitectura de software (Hexagonal/Ports & Adapters)
- Persistencia de datos (JPA, JDBC)
- Procesamiento de datos (Spring Batch)
- Integración con APIs externas
- Validación de datos (XSD, XML)
- Consultas avanzadas en PostgreSQL

## Requisitos Comunes

- Java 21 (LTS)
- Maven 3.9+
- PostgreSQL 16
- Spring Boot 3.2+
